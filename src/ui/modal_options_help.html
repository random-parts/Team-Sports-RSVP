Extra Options Help Menu

  • Re-install Team Form

    This will remove the Form attached to the spreadsheet and create a new Google Form; setting the spreadsheet as its destination.

    The old Form file with still be inside Google Drive, and will have to be manually trashed.

  • Re-install Triggers

    Re-install the project triggers for this spreadsheet.

  • Re-set Web Link

    Add the weblink manually in-case the wrong one was entered, or just because it's kinda fun to do.

    If there are values within the schedule ranges it will wipe the sheet in order to give the new link data a fresh start. This will also remove the values within the rsvp ranges.

  • Remove Conditional Formatting

    Removes the conditional formatting within the squad range.

  • Remove Dependencies
    [soft un-install]

    This is a bit of a soft-uninstall. It will remove the triggers, the document properties[settings], the template sheet, and the Form response sheet while also detaching the form.

  • Remove Triggers

    Remove the Triggers for this spreadsheet if you prefer this add-on doing absolutely nothing. Triggers will re-install if a new season is created and they do not exist.

    Removing the add-on will remove all project triggers across all of the [user owned] spreadsheets this add-on was used with.

  • Restore Conditional Formatting

    Restores the conditional formatting within the squad range; in case it had been tragically removed.

  • Restore Data Validation

    This will reset the data validation preventing uncouth and malicious input.

  • Restore Named Ranges

    If the NamedRanges get too screwy from row and column deletes, resetting them here should help. Maybe.
